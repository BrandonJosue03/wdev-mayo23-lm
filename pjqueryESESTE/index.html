API

Al presionar el boton "API" te redireccionara al API general que tendra la base de datos con la que vamos a trabajar

Crear Curso

Al presionar el boton "Crear Curso" te redireccionara a una página donde podras agregar el nombre, la descripcion y el tiempo que va a estar el curso creado.

Listar Curso

Al presionar "Listar Curso" te redireccionara a la pagina donde puedes ver tu curso creado y ademas ver el de tus compañeros, podrás editar cualquier curso desde ésta página.
